Troubleshooting

If you are experiencing issues with Skillify Canvas, please refer to the following troubleshooting steps to resolve common problems:

Login Issues

If you are having trouble logging in:

  • Make sure you are using the correct username and password.
  • Check your internet connection.
  • Clear your browser's cache and cookies.
  • Contact our support team if the issue persists.

Page Loading Problems

If web pages are not loading properly:

  • Refresh the page.
  • Try a different web browser.
  • Ensure JavaScript is enabled in your browser.
